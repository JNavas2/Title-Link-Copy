Title-Link Copy

Advanced copying of web page titles, URLs, and rich text hyperlinks.
Access these features via the Right-Click Menu or Keyboard Shortcuts.

Functionality:

  • Page background → Page Title + Page URL
  • Link context → Link Text + Link URL
  • Option: Selected Text inclusion
  • Option: AP-style Title Case formatting

Formats titles per AP Stylebook rules (e.g., "this is a test" → "This Is a Test").
